Làm thế nào để lấy API key?
Google Gemini AI:
  1. Truy cập Google AI Studio
  2. Đăng nhập bằng tài khoản Google của bạn
  3. Chọn "Get API key" hoặc "Create API key"
  4. Sao chép API key và dán vào ô bên trái
OpenAI:
  1. Truy cập OpenAI Platform
  2. Đăng nhập tài khoản OpenAI
  3. Tạo Secret key mới và sao chép
Anthropic:
  1. Truy cập Anthropic Console
  2. Đăng ký tài khoản nếu chưa có
  3. Tạo API key mới
